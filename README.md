
<h1>LOADING SCREEN</h1>
<p>Improves pre round loading screen by teleporting players to random locations in the map</p>
<p>Config:</p>
<ul>
<li>ls_doorId</li>
</ul>
<p>Type:</p>
<ul>
<li>String List</li>
</ul>
<p>Description:</p>
<ul>
<li>Doors id of doors to use as teleport point</li>
</ul>
<p>Default:</p>
<ul>
<li>* ( means every door in the facility )</li>
</ul>
